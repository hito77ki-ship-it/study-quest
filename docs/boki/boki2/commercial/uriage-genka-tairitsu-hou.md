# 【2級・商簿】売上原価対立法

## 1. コア概念（一言で言うと）
商品売買を「商品」（資産）・「売上原価」（費用）・「売上」（収益）の3勘定で記帳し、販売のたびに売上原価を即座に確定させる方法。三分法と異なり期中仕訳の時点で売上原価が分かるため、決算整理仕訳（しくりくりし）が不要になる。日商簿記2級の教科書では「参考マーク」（出題可能性が低い論点）付きで紹介されている。

## 2. 標準的な仕訳パターン、または計算プロセス
**仕入時：** 商品という資産を取得したと捉える
```
（借）商品　×××　（貸）現金（買掛金等）　×××
```

**売上時：** 売上の計上と同時に、販売した商品の原価を「商品」から「売上原価」へ振り替える（2行の仕訳を同時に行う）
```
（借）売掛金（現金等）　×××　（貸）売上　×××
（借）売上原価　×××　（貸）商品　×××
```

**決算時：** 期中仕訳の時点で勘定残高がすでに財務諸表計上額になっているため、三分法のような決算整理仕訳（しくりくりし）は不要。

## 3. 執筆時の必須キーワード・引っかかりやすい落とし穴
- **「なぜ仕入勘定を使わず売上原価勘定を新設するのか」が読者の最大の疑問点**：「仕入」は期中に買った金額の合計を表す勘定であり、"売れた分の原価"を表す勘定ではないため、と説明する（本サイトの記事では家計簿アプリ／コンビニPOSレジのたとえ話で解説済み）
- 三分法との対比を必ず入れる：三分法＝期末にまとめて売上原価を算定（決算整理必要）、売上原価対立法＝取引のたびにリアルタイムで算定（決算整理不要）
- 2級教科書では出題可能性が低い「参考」論点として扱われている点に触れておくと、読者の「これは覚えるべきか」という不安に答えられる
- 商品を仕入れた時点では「商品」（資産）が増えるだけで費用計上されない、費用（売上原価）になるのは売れた時点、という資産と費用のタイミングのズレが理解のポイント

> 売上原価対立法は、商品売買について、「商品」勘定（資産）、「売上原価」勘定（費用）および「売上」勘定（収益）を用いて記帳する方法です。三分法と異なり、期中仕訳で売上原価を算定するため、売上原価対立法では決算整理仕訳は行いません。
>
> ⑵仕入時の会計処理
> 売上原価対立法では、商品の仕入取引を「商品という資産の取得」と捉えます。そのため、仕入時に「商品」勘定（資産）の増加として処理します。

> （借）商品［資産＋］900　（貸）現金［資産－］900
> ※商品という資産を取得したと考えるため、「商品」勘定の増加
