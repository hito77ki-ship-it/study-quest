# 【1級・商会】総合償却と個別償却

## 1. コア概念（一言で言うと）
複数の固定資産をまとめて1つの償却単位として減価償却費を一括計算する方法が総合償却。資産ごとに個別に計算する通常の方法（個別償却）と対比される、実務上の効率化のための計算方法。

## 2. 標準的な仕訳パターン、または計算プロセス
総合償却の計算は必ず以下の順序で行う（順序を変えると金額が変わるため厳守）。

1. **①要償却額合計を算定**：（各資産の取得原価合計－残存価額合計）
2. **②各資産の毎年の減価償却費（個別計算ベース）の合計額を算定**
3. **③平均耐用年数を算定**：①÷② → 1年未満の端数は切り捨て
4. **④総合償却費を算定**：①÷③（②の数字は再利用しない、①を③で割り直す）

除却時の仕訳（除却資産の要償却額を減価償却累計額から取り消す）：
```
（借）減価償却累計額　×××　（貸）固定資産　×××
　　　貯蔵品　　　　　×××
```
※当初の残存価額と処分可能見込額が同じ場合は「固定資産除却損」を計上しない。

## 3. 執筆時の必須キーワード・引っかかりやすい落とし穴
- **④の計算で②の数字をそのまま使ってしまうミスが最頻出**：②はあくまで③（平均耐用年数）を求めるための一時的な数字。最終的な総合償却費は必ず①÷③で計算し直す
- **平均耐用年数の端数処理は「切り捨て」**（四捨五入ではない）
- 総合償却では個々の資産の正確な未償却残高が分からないため、除却時に「固定資産除却損」を計上できない（個別償却との決定的な違い）
- 総合償却は「正確さより実務上の効率」を優先した方法、という設計思想を先に伝えると納得感が出る（工場に同種の機械が何十台もある場合など）

## 4. テキスト原文・参照データ
出典：日商簿記1級 商業簿記・会計学（登川）教科書 第Ⅰ部 第4章第3節「7 総合償却」（p.89）

> 総合償却とは、複数の固定資産を一つの償却単位として、一括して減価償却費を計算する方法をいう。
>
> ⑵計算方法
> ①償却単位全体の要償却額（取得原価合計－残存価額合計）を算定
> ②各資産の毎年の減価償却費の合計額を算定
> ③平均耐用年数を算定　※償却単位全体の要償却額（①）÷各資産の毎年の減価償却費の合計額（②）＝平均耐用年数
> ④減価償却費を算定　償却単位全体の要償却額（①）÷平均耐用年数（③）＝減価償却費
> ※平均耐用年数の端数処理により②と④の減価償却費は異なる。よって、総合償却による減価償却費の計算は、必ず上記の順序で行うこと。
>
> ⑶除却時の処理
> 除却した場合は、除却資産の要償却額（取得原価－残存価額）を「減価償却累計額」勘定から控除する。
> （借）減価償却累計額×××　貯蔵品×××　（貸）固定資産×××
> ※当初の残存価額と処分可能見込額が同じである場合には、「固定資産除却損」勘定を計上しない。

例題13（教科書p.89〜90）の数値例：Ａ機械5,000千円(耐用5年)・Ｂ機械5,000千円(耐用6年)・Ｃ機械8,000千円(耐用12年)・Ｄ機械10,000千円(耐用9年)、残存価額はいずれも10%。
> ①要償却額の合計：（5,000＋5,000＋8,000＋10,000）×0.9＝25,200（千円）
> ②年間償却額の合計：Ａ機械5,000×0.9÷5年＝900／Ｂ機械5,000×0.9÷6年＝750（以下Ｃ・Ｄ機械も同様に算定し合算）
