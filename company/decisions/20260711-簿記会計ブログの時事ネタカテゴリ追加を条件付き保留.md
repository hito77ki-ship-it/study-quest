# 簿記・会計ブログへの時事ネタカテゴリ追加

**日付：** 2026年7月11日
**決定者：** CEO ジョブズ / 取締役
**ステータス：** 条件付き保留

## 背景

取締役が別セッション（Claudeアプリ）で「簿記会計ブログに時事ネタ（インボイス制度・電子帳簿保存法の改正等）も扱うべきか」を相談し、その回答を持ち帰って本セッションに共有した。Claudeアプリ側の分析は「検索流入が跳ねやすい」「網羅型記事との差別化になる」「税理士志望者としての実務力にもなる」というメリットと、「鮮度管理の手間」「政治的な踏み込みすぎへの注意」というリスクを挙げ、カテゴリを分けて追加する方向を示唆していた。

## リサーチ・検証内容

このセッションで以下を実地確認した。

1. **既存カバー状況：** `インボイス制度`・`電子帳簿保存法`は現在サイト内に該当記事が1本もなく、着手すれば完全新規カテゴリになる。
2. **直近の公開ペース：** 過去7日間で新規記事15本を公開しており、ほぼ毎日2本以上のハイペースで網羅型記事の量産が進行中。
3. **インデックス状況：** GSCの手動インデックス登録リクエストキャンペーン（[[project_seo_indexing_baseline]]）はDay1（12件）を終えたのみで、Day2は本日中に着手する時間が取れず翌日に持ち越し。199記事のうち大半がまだGoogleに評価されきっていない段階。
4. 過去に同種の提案が`company/decisions/`に記録されていないか確認したが、初出のテーマだった。

## ジョブズの所見

Claudeアプリの分析は技術的には妥当だが、**この会社の現在のキャパシティを見ていない**という弱点がある。

- **鮮度管理の負債化リスク：** 時事ネタは「公開して終わり」ではなく、制度改正のたびに追いかけて更新し続ける継続コストが発生するジャンル。現状ですら網羅型記事を週15本ペースで出しながらGSC申請すら消化しきれていない一人運営の会社に、更新責務を伴う新カテゴリを足すのは、既存の首を絞めるだけで終わる可能性が高い。
- **YMYL的リスクの過小評価：** 税制解説は「お金に関わる正確性」がGoogleの評価・信頼性に直結するジャンル。Claudeアプリ側の指摘は「政治的に踏み込みすぎるな」止まりだが、本質的なリスクは政治色より「情報が古いまま放置され、間違った税務情報が残ること」の方が実害が大きい。更新を継続できない前提でこのジャンルに手を出すのは危険。
- アイデア自体は悪くない。差別化・実務力の両面で妥当性はあり、将来的には検討に値する。

「それはユーザーの人生を本当に変えるか？」で見ると、今のボトルネックは「コンテンツの種類が足りない」ことではなく、「既存199本がまだインデックス・評価されきっていない」こと。ここで手を広げるのはシンプルさの原則にも反する。

## 影響

- 現時点でのアクションなし。新カテゴリは着手しない。

## フォローアップ（着手条件）

以下がそろった時点で再提案してよい：
- [ ] GSCインデックス登録キャンペーン（192〜199本一巡、約16日計画）が完了し、既存記事の評価がある程度進んだ状態になったとき
- [ ] 網羅型記事の新規公開ペースが落ち着き、更新を継続できる余力が生まれたとき（目安：新規公開が週数本程度に落ち着いた頃）
- 着手する場合も、まず1〜2本の試験投稿で更新運用が回るかを検証してから本格展開する

## 参考：着手時のリサーチ手法（2026-07-11追記）

着手条件が揃った際の運用案として、取締役と検討した内容を記録しておく。

**推奨ツール：Perplexity**（Deep Researchモード）。ChatGPT Deep Research・Gemini Deep Research・Claudeとの比較を検索で確認したところ、リアルタイムのWeb検索インデックスを持ち鮮度重視の時事情報に強く、独立監査（Columbia大学Tow Center）でも引用失敗率が他のAI検索エンジンより低いという結果だった。ChatGPT Deep Researchは長時間（最大30分）かけた構造化レポートが得意、Claudeは文章としての完成度が高い、という住み分け。推奨フローは「Perplexityで一次情報を集める→Claude Codeで記事化」。

**注意点：** Deep Researchの引用自体にも捏造リスクがある（2026年に入り学術論文での捏造引用が急増という調査結果あり）ため、出典を鵜呑みにせず国税庁・財務省等の一次情報を直接開いて確認する一手間は必須。これは既存の「CPA教科書PDFでの公開前照合」の習慣（[[reference_cpa_learning_textbooks]]）と同じ運用で対応できる。

**リサーチ用プロンプト案：**
```
[トピック名、例：インボイス制度の2027年改正]について、以下の観点でDeep Researchしてください。

1. 制度変更の正式名称と根拠法令（条文番号・公布日があれば含める）
2. 施行日・適用開始日（すでに確定しているか、検討段階かを明記）
3. 変更点の具体的内容（変更前とのbefore/after比較）
4. 対象となる事業者・個人の条件
5. 具体的な数値基準（金額・割合・期間等）は正確に
6. 一次情報源（国税庁・財務省・中小企業庁・e-Gov等の公式ページ）へのリンクを必ず明記し、ニュースサイトやまとめブログなど二次情報だけを根拠にしないこと
7. 情報の確定度合い（施行済み／閣議決定済み／審議中の法案／観測記事レベル、のどれか）を明記すること
8. この記事は日商簿記1級・税理士試験の受験生向けブログの解説記事の下敷きにします。試験範囲との関連があれば触れてください

推測や未確定の観測は断定的に書かず、一次情報を優先してください。
```
