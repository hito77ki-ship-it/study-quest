# ゼロから簿記シリーズ（boki-zero-01〜05）のファクトチェック結果

**日付：** 2026年7月7日
**決定者：** CEO ジョブズ / 取締役承認
**ステータス：** 採用

## 背景

取締役が、自身の大学の簿記研究部の新入生（日商簿記3級学習者）にサイトを紹介する予定があるとのこと。3級初心者の入口となる「簿記3級をゼロから始める」シリーズ（boki-zero-01〜05.html、全5本）は、Tier1・Tier2いずれの計画にも含まれない未チェックの記事群だったため、紹介前に急遽ファクトチェックを実施した。

## リサーチ要点

5本を並行フォークでファクトチェック。

| 記事 | 結果 |
|------|------|
| 第1回（お店の取引とは？簿記とは？） | 2件の問題あり（下記） |
| 第2回（なぜ簿記が大事？） | 問題なし |
| 第3回（簿記の流れ・決算書とは？） | 問題なし |
| 第4回（仕訳のしくみ・借方と貸方） | 本文は問題なし |
| 第5回（決算とは？） | 問題なし |

**boki-zero-01.htmlで見つかった問題：**

1. **タイトル・meta descriptionと本文の不一致：** `<title>`・meta descriptionは「仕訳とは？借方・貸方の意味を解説」となっていたが、実際の本文は「取引とは？簿記とは？」のみで、仕訳・借方・貸方の内容は第4回（boki-zero-04.html）にしか存在しなかった。検索結果を見てクリックした読者が期待と違う内容に着地する、実害のあるSEO/UXバグ。og:title・og:descriptionは本文と一致しており正しかった。
2. **合格率の楽観的表現：** FAQ内で「日商簿記3級の合格率はおおむね40〜50%」と2箇所に記載。直近実績は統一試験平均約35.5%、ネット試験（2025年4〜12月）40.8%で、実態は35〜43%程度。Tier1計画時に指摘した「合格率が実態より楽観的」という典型パターンの再発。

副次的に、FAQの中に本文で扱っていない「仕訳の借方・貸方の覚え方」という質問が紛れており、これも本文との不一致（同根の問題）だったため、本文の実際のトピック（「取引」の定義）に沿った質問に差し替えた。

Sources: [日商簿記検定試験（２級・３級）ネット試験 受験者データ（2025年4月〜12月）](https://www.kentei.ne.jp/50078)／[簿記3級の合格率は？（フォーサイト）](https://www.foresight.jp/boki/column/passrate/)

## 決定内容

boki-zero-01.htmlを以下の通り修正・実行済み：

- `<title>`・meta descriptionを本文の内容（お店の取引とは？簿記とは？）に合わせて修正
- FAQ「仕訳の借方・貸方の覚え方は？」を、本文の内容に沿った「簿記でいう『取引』とは何ですか？」に差し替え
- 合格率の記載を「40〜50%」から「35〜40%程度」（1つ目）、「統一試験で30%台半ば、ネット試験で40%前後」（2つ目）に修正
- dateModifiedを更新

他4本は修正不要。

## ジョブズの所見

「40〜50%」というたった数文字のズレでも、本番の合格率を35%と予想していた受験生が「50%なら余裕」と油断する引き金になり得る。地味だが、これこそ「間違った情報のまま勉強を進めて本番で戸惑う」を防ぐという今回のTier1計画の初志そのもの。

タイトルと本文のズレは、後輩に紹介する直前に見つかったのが不幸中の幸い。もし気づかず紹介していたら、「仕訳の解説だと思って読んだのに違う話だった」という第一印象で離脱されていたかもしれない。第一印象で信頼を失うのが一番もったいない。

## 影響

- 検索結果での期待とのズレによる離脱リスクを解消
- 合格率の記載が実態に近づき、受験生の期待値が適正化される
- 大学の簿記研究部への紹介を、より安心して進められる状態になった
