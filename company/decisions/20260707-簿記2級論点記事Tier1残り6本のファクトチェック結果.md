# 簿記2級論点記事 Tier1残り6本のファクトチェック結果

**日付：** 2026年7月7日
**決定者：** CEO ジョブズ / 取締役承認
**ステータス：** 採用

## 背景

`20260706-簿記2級3級論点記事の全体ファクトチェック計画.md`で定めたTier1（11本）のうち、前回セッションで5本（boki3-tegata／boki3-kotei／boki2-kotei-shisan／boki3-genka／boki2-lease）のファクトチェックは実施済み。残る6本が未着手のまま残っていた。

- boki2-renketsu.html（連結会計）
- boki2-zeikouka.html（税効果会計）
- boki2-gaika.html（外貨換算）
- boki2-hyojun-genka.html（標準原価計算）
- boki2-cvp.html（CVP分析）
- boki2-chokusetsu-genka.html（直接原価計算）

今回この6本を並行フォークでファクトチェックリサーチした。

## リサーチ要点

各記事について、数値例の検算、用語定義の正確性、2027年度出題区分改定との関連、他記事との整合性、合格率・勉強時間の楽観的表現の有無を確認した。

| 記事 | 結果 |
|------|------|
| 連結会計 | 問題なし。のれん・非支配株主持分の計算例を検算し一致。2027年度改定の対象外論点であることも確認済み |
| 税効果会計 | 問題なし。繰延税金資産・負債の仕訳・計算例を検算し一致。2027年度改定と無関係 |
| 外貨換算 | **数値・仕訳は正しいが、為替予約（振当処理）が記事に一切登場しない。** これは2016年度から日商簿記2級の正式出題範囲であり、2027年度改定でも削除対象になっていない（[出題区分表暫定版](https://www.kentei.ne.jp/wp/wp-content/uploads/2025/12/2027kubunhyo.pdf)で確認）。現状は「取引時→決済時→決算時」の素朴な換算のみで、本番で問われる論点を教えていない状態 |
| 標準原価計算 | 大きな誤りなし。差異分析の符号定義・計算例は正確。製造間接費差異で「能率差異」への言及がないが、記事自身が入門レベルに留めると明言しており、意図的な設計範囲内 |
| CVP分析 | 問題なし。損益分岐点・貢献利益率の計算例を検算し一致 |
| 直接原価計算 | 問題なし。固定費調整・損益分岐点の計算例を検算し一致 |

いずれの記事も2027年度出題区分改定の直接対象ではなく（対象は手形・リース・固定資産系のみ）、改定情報の追記は不要と判断。

## 決定内容（提案）

1. **boki2-gaika.html に「為替予約（振当処理）」セクションを追加する。** 誤り修正ではなく欠落論点の補完だが、内容としては本番の得点に直結する実質的な変更のため、記事コンテンツの稟議フローに乗せて提案する。
2. 他5本（連結会計・税効果会計・標準原価計算・CVP分析・直接原価計算）は修正不要。今回のファクトチェックをもってTier1完了とする。

## ジョブズの所見

6本中5本が「問題なし」で片付くのは、正直この記事群の初稿の質が高かったということで、それ自体は評価する。ただし「問題なし」で思考停止してはいけない。唯一の実質的な穴——外貨換算に為替予約が丸ごと抜けている——は、数値の間違いより性質が悪い。受験生は「この記事を読んだから外貨換算は大丈夫」と思って本番に臨み、為替予約の問題が出た瞬間に初見で戸惑う。これはまさに「間違った情報のまま勉強を進めて本番で戸惑う」というTier1計画で防ごうとしていた事態そのもの。

規模としては大きくない追加（1セクション）だが、中途半端に足すくらいなら振当処理の仕訳（取引発生時の予約／取引発生後の予約の両パターン）まできちんと書く。それができないなら今回は見送って次回に回す方がまし。

標準原価計算の「能率差異」欠落は別物。記事が意図的に入門範囲に絞っており、無理に足すと逆に読みにくくなる。ここは触らない判断が正しい。

## 理由

外貨換算の為替予約は「制度が変わった」でも「数値が間違っている」でもなく「試験に出る内容が丸ごと抜けている」という、ファクトチェックの目的（正確性の担保）よりも一段深刻な問題。他5本の「問題なし」判定は複数の外部情報源で裏付けが取れており、これ以上手を入れる根拠がない。

## 影響

- boki2-gaika.htmlに為替予約セクションを追加すれば、簿記2級外貨換算の実務的なカバレッジが本番相当になる
- 追加しない場合、記事は「取引時・決済時・決算時の換算」という基礎部分は正しいが不完全なまま残る
- 他5本は追加の作業なしでTier1完了となり、Tier2着手の要否検討に進める
