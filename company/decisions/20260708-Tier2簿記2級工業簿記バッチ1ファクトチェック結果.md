# Tier2 簿記2級工業簿記バッチ1（7本）ファクトチェック結果

**日付：** 2026年7月8日
**決定者：** CEO ジョブズ / 取締役承認
**ステータス：** 採用

## 背景

Tier2の2級工業簿記バッチ1（工業簿記概論／材料費／労務費／経費／製造間接費／部門別原価計算／個別原価計算）7本を並行フォークでファクトチェックした。用語・解法の基準としてCPAラーニング（取締役が普段使用）を優先参照先に指定した。

## リサーチ要点

| 記事 | 結果 |
|------|------|
| boki2-kogyo.html（工業簿記概論） | 誤り：「配点上も工業簿記は大きく」と記載しているが事実は逆。2級の配点は商業簿記60点・工業簿記40点で商業簿記の方が大きい |
| boki2-zairyouhi.html（材料費） | 問題なし |
| boki2-roumuhi.html（労務費） | 誤りなし。ただし賃金消費額の計算式・賃率差異のセクションに具体的な数値例が抜けている |
| boki2-keihi.html（経費） | 誤り：「工場消耗品費」を経費として分類しているが、同記事内の別表では正しく材料費（間接材料費）に分類しており自己矛盾。正しくは材料費 |
| boki2-seizo-kansetsuhi.html（製造間接費） | 問題なし。シュラッター図の数値もすべて検算一致 |
| boki2-bumonbetsu.html（部門別原価計算） | 問題なし。直接配賦法・相互配賦法とも配賦表の数値がすべて検算一致し、記事群の中でも計算精度が特に優れている |
| boki2-kobetsu-genka.html（個別原価計算） | 問題なし |

Sources: [日商簿記2級の配点を知れば合格が見える！（経理実務の学校）](https://edu.jusnet.co.jp/edu_article/certification/bookkeeping_article/47313/)／[材料費とは？（いぬぼき）](https://inuboki.com/2q-koubo-kouza/koubo-kouza1-1/)

## 決定内容

1. `boki2-kogyo.html`：配点の記述を「商業簿記60点・工業簿記40点」の事実に修正
2. `boki2-keihi.html`：「工場消耗品費」を経費の具体例から削除し、材料費（間接材料費）に統一
3. `boki2-roumuhi.html`：賃金消費額の計算例、賃率差異の計算例をそれぞれ追加

## ジョブズの所見

工業簿記は3級・2級商業簿記に比べて誤り率が明確に低かった（7本中3本）。特に部門別原価計算は「記事群の中でも計算精度が特に優れている」という評価で、直接配賦法・相互配賦法の端数処理まで作り込まれていた。配賦計算のような機械的な計算は、AIが得意とする領域なのだと思う。

一方で見つかった3件は、いずれも「事実確認を怠った」タイプの誤り——配点比率、勘定科目の分類——で、計算ミスとは性質が違う。これは検算では見つけられず、外部情報源との照合でしか見つからない種類のミス。CPAラーニングを基準に指定したことも、こうした事実確認の精度向上に寄与したと考えられる。

## 影響

- 工業簿記概論・経費の事実誤認が解消され、初学者が誤った前提で学習を進めるリスクがなくなる
- 労務費の計算例追加により、頻出論点（賃率差異）の実践的な理解が深まる
