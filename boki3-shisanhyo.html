合計試算表・合計残高試算表・残高試算表
3つの違いと書き方を完全解説

著者：大谷 一輝（大阪経済大学3回生・日商簿記1級勉強中）

「仕訳は合っているのに、答案用紙の形式を間違えて0点」——試算表で一番やりがちなミスです。
3種類の試算表は答案用紙を見た瞬間に「合計欄か残高欄か」を判断できれば怖くありません。この記事で違いを整理しましょう。

簿記の2大帳簿：仕訳帳と総勘定元帳

試算表を理解する前に、取引を記録する2つの主要簿を知っておきましょう。

仕訳帳（主要簿①）
取引を日付順に借方・貸方で記録する帳簿。「いつ・どんな取引があったか」を日記のように記録する。
総勘定元帳（主要簿②）
勘定科目ごとに金額を集計するT字型の帳簿。「現金の残高はいくらか」などを科目別に確認できる。
流れ：仕訳帳 → 転記 → 総勘定元帳 → 試算表
仕訳帳が「原本」、総勘定元帳が「科目別インデックス」。仕訳帳の各取引を科目別に元帳へ移す作業を「転記」と呼びます。

転記とは？（4ステップ）

仕訳帳の取引1行を、対応する2つの元帳ページへ転記します。

  1. 借方の科目（例：現金）の元帳ページを開く
  2. 現金勘定の借方欄に金額を記入
  3. 貸方の科目（例：売上）の元帳ページを開く
  4. 売上勘定の貸方欄に金額を記入 → 完了！
転記のルール：方向は変わらない
仕訳帳の借方科目 → 元帳の借方へ、貸方科目 → 元帳の貸方へ。仕訳と同じ方向にそのまま転記するだけです。

試算表を作る目的

試算表は「転記が正確に行われているか」を検証するための一覧表です。仕訳帳に書かれた仕訳を総勘定元帳へ転記（科目別に写す作業）した後、全科目の残高を一覧化して借方合計と貸方合計が一致するか確認します。

転記の流れ
仕訳帳（日付順に記録）→ 転記 → 総勘定元帳（科目別に集計）→ 試算表（全科目の一覧）
試算表で発見できること
転記ミス・記入漏れ・計算ミスを早期発見できる。借方合計 ≠ 貸方合計 → 必ずどこかにミスあり。
試算表と決算の関係
試算表は期中に何度でも作成できます。決算では「決算整理前残高試算表（前T/B）」を作成し、減価償却などの決算整理仕訳を行った後に「決算整理後残高試算表（後T/B）」を作成します。最終的にこの後T/Bを元に B/S・P/L が完成します。

決算の5ステップと試算表の位置づけ

試算表は決算プロセスのインプットとして使われます。決算の全体像を先に把握しておきましょう。

  1. 残高試算表（前T/B）作成：期末時点の各科目残高を集計
  2. 決算整理仕訳：減価償却・貸倒引当金など修正仕訳（→第12章）
  3. 残高試算表（後T/B）作成：修正後の残高を集計
  4. 帳簿の締め切り：収益・費用勘定をゼロリセット（→第11章）
  5. 財務諸表（B/S・P/L）作成：決算のゴール！
試算表は「前T/B」と「後T/B」の2回作成する
期中に何度でも作成でき（月次・四半期も可）、ミスの早期発見に役立ちます。

3種類の試算表：まず違いをつかむ

【合計】が付く場合
借方・貸方それぞれの合計金額を両方記入する。取引の総額がわかる。
【残高】が付く場合
借方と貸方の差引した残高のみ、どちらか一方に記入する。
タイプA

合計欄のみ
タイプB

合計と残高の両方を記入する。一番情報量が多い。

試験での確認ポイント：答案用紙を最初に見る！
問題を解く前に必ず答案用紙の列ヘッダーを確認する。「合計」と書いてあれば両方記入、「残高」と書いてあれば一方のみ。これだけで形式ミスはゼロになります。

よくある質問

残高試算表と合計試算表の違いは何ですか？

合計試算表は各勘定の借方合計・貸方合計を集計した表です。残高試算表は差額（残高）だけを集計した表です。合計残高試算表は両方を合わせたもので、3種類とも試験範囲です。

試算表の借方と貸方はなぜ必ず一致するのですか？

複式簿記では必ず借方と貸方が同額で記録されるため、全取引を集計した試算表では借方合計と貸方合計が必ず一致します。一致しない場合は転記ミスや計算間違いがあります。

試算表と精算表の違いは何ですか？

試算表は帳簿の記録を確認するための表です。精算表は決算整理を行いP/L・B/Sを作成するための作業表です。試算表を精算表のインプット（残高試算表欄）として使います。
